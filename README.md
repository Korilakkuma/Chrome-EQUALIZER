Chrome EQUALIZER
=========

Chrome extension for Graphic Equalizer.  
This uses Web Audio API Library ([XSound](https://github.com/Korilakkuma/XSound)).

## 1. Get repository

```bash
$ git clone git@github.com:Korilakkuma/Chrome-EQUALIZER.git
```

## 2. Install npm packages

```bash
$ cd Chrome-EQUALIZER
$ npm install
```
